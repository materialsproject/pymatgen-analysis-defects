A class representing the a harmonic defect vibronic state.

The vibronic part of a defect is often catured by a simple harmonic oscillator. This class store a representation of the SHO as well as some additional information for book-keeping purposes.

Create a HarmonicDefect from a list of vasprun.

Note

The constructor check that you have the vaspruns sorted by the distortions but does not perform the sorting for you. This serves as a safety check to ensure that the vaspruns are properly ordered.

Parameters:

  • vaspruns (list[Vasprun]) – A list of Vasprun objects.

  • charge_state (int) – The charge state for the defect.

  • kpt_index (int) – The index of the kpoint that corresponds to the band edge.

  • spin_index (Optional[int]) – The index of the spin that corresponds to the band edge. If None, we will assume that the band edge is spin-independent and we will use the spin channel with the most localized state.

  • relaxed_index (Optional[int]) – The index of the relaxed structure in the list of structures.

  • defect_band_index (Optional[int]) – The index of the defect band (0-indexed). This is found by looking at the inverse participation ratio of the different states.

  • procar (Optional[Procar]) – A Procar object. Used to identify the defect band if the defect_band_index is not provided.

  • store_bandstructure (bool) – Whether to store the bandstructure of the relaxed defect calculation. Defaults to False to save space.

  • get_band_structure_kwargs (Optional[dict]) – Keyword arguments to pass to the get_band_structure method.

  • **kwargs – Additional keyword arguments to pass to the constructor.

Return type:

HarmonicDefect

get_elph_me(wswqs)[source]#

Calculate the electron phonon matrix elements.

Combine the data from the WSWQs to calculate the electron phonon matrix elements. The matrix elements are calculated by combining the finite difference from the matrix overlaps.

d(<W|S|W(Q)>) / dQ

And the eignvalue difference.

Parameters:

  • wswqs (list[WSWQ]) – A list of WSWQ objects, assuming that they match the order of the distortions.

  • bandstructure – The bandstructure of the relaxed defect calculation.

Returns:

The electron phonon matrix elements from the defect band to all other bands.

The indices are: [spin, kpoint, band_i, band_j]

Return type:

npt.NDArray
